# 二.认识Linux

## 1.linux 与 window 不同

- Linux 严格区分大小写
- Linux 中所有的内容以文件形式保存，包括硬件、用户和文件
- Linux 不考扩展名区分文件类型，是靠权限来区分，但是有一些约定的扩展名，是给管理员看的
  - 压缩包`.gz`、`.bz2`、`.tar.bz2`、`.tgz`
  - 二进制文件`.rpm`
  - 网页文件`.html`、`.php`
  - 脚本文件`.sh`
  - 配置文件`.conf`
- Linux 更多使用字符界面
- 占用的系统资源更少
- 减少了出错和被攻击的可能性，会让系统更稳定
